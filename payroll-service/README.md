# payroll-service

The `payroll-service` is responsible for payroll processing within NexusHR.

## Purpose

- Manage salary-related operations
- Handle payroll calculations
- Store payroll records and related business logic

## Port

- `8083`

## Technology

- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Flyway
- Validation

## Run

From the `payroll-service` folder:

```bash
mvnw.cmd spring-boot:run
```

## Planned Responsibilities

- Salary calculation
- Payroll record creation
- Payroll history tracking
- Secure payroll access for authorized roles
